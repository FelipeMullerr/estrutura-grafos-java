# Projeto de Grafos em Java

Este projeto implementa estruturas e algoritmos clássicos de grafos em Java, permitindo a criação, manipulação e análise de grafos utilizando listas ou matrizes de adjacência. O grafo pode ser criado manualmente ou a partir de um arquivo `.txt` seguindo um formato específico.

## Funcionalidades

- Criação de grafos direcionados/não direcionados e ponderados/não ponderados
- Criação e Remoção de Arestas
- Representação por lista ou matriz de adjacência
- Inserção e remoção de vértices e arestas
- Busca em largura (BFS) e profundidade (DFS)
- Algoritmo de Dijkstra para caminhos mínimos (em grafos ponderados)
- Leitura de grafos a partir de arquivo

## Observações

- Para grafos não direcionados, as arestas são inseridas nos dois sentidos.
- Para grafos ponderados, o peso deve ser um número real (ex: `2.5`).
- O projeto suporta tanto listas quanto matrizes de adjacência

## Informações

- Projeto criado para a disciplina de Grafos do curso de Ciência da Computação - Univali
